# 移位运算

## 移位运算的意义

15m = 1500cm

**机器用语：15 相对于小数点左移两位**

> 计算机是没有小数点的表示

## 算术移位的规则

左移一位机器数对应的真值绝对值变为原来的二倍

**符号位不变**

|      | 码制             | 添加补码      |
| ---- | ---------------- | ------------- |
| 正数 | 原码、补码、反码 | 0             |
|      | 原码             | 0             |
| 负数 | 补码             | 左移：右侧添0 |
|      |                  | 右移：左侧添1 |
|      | 反码             | 1             |
